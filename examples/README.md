# Reference Projects

Related AI agent projects. Clone these here and use as reference when needed.

| Project | Link | Summary |
|---------|------|---------|
| **pi-mono** | https://github.com/badlogic/pi-mono | AI agent toolkit by Mario Zechner. Coding agent CLI, unified LLM API, TUI and web UI libraries, Slack bot. Extensible via skills, prompt templates, and TypeScript extensions. |
| **OpenCode** | https://github.com/anomalyco/opencode | Open source AI coding agent. Built with Go, Bubble Tea TUI, multi-provider support, session management, LSP integration. Very similar to Claude Code in capability. |
| **OpenClaw** | https://github.com/openclaw/openclaw | Personal AI assistant by Peter Steinberger. Multi-channel (WhatsApp, Telegram, Slack, Discord, etc.), multi-agent routing, voice, skills, cron, heartbeat. Node.js gateway. |
| **nanobot** | https://github.com/HKUDS/nanobot | Ultra-lightweight OpenClaw (~4k lines of Python). Core agent loop, memory, skills, subagents, multi-channel chat, cron, heartbeat. Research-ready and readable. |
| **nanoclaw** | https://github.com/qwibitai/nanoclaw | Minimal OpenClaw-inspired agent. |
| **Moltis** | https://github.com/moltis-org/moltis | Personal AI assistant built in Rust. Single binary, multi-provider LLMs, long-term memory, sandboxed execution (Docker/Apple Container), MCP tools, web UI. |
| **zeptoclaw** | https://github.com/qhkm/zeptoclaw | Minimal claw-family agent. |
| **ironclaw** | https://github.com/nearai/ironclaw | NEAR AI's claw-family agent. |
